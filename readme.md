## Overview

**AeraChat** is a modern real-time chat application built using the MERN stack and Socket.io.
It enables users to communicate instantly with a smooth and responsive user interface.

The application focuses on real-time messaging, authentication, and seamless user experience.

> Chat Anytime, Anywhere

---

## Website Details

* User Authentication System
* Real-Time Messaging with Socket.io
* Online/Offline Status Indicator
* User Search Functionality
* Fully Responsive UI
* Modern Dark-Themed Interface

The application is designed to provide a fast and interactive chatting experience with clean UI and scalable backend architecture.

---

## Key Features

* ✅ Secure User Registration & Login (JWT Authentication)
* ✅ Real-time messaging using WebSockets
* ✅ Online/Offline presence detection
* ✅ Persistent chat history (MongoDB)
* ✅ Private one-to-one conversations

## Installation & Setup

Follow the steps below to run the project locally:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Ashiqur1819/AeraChat.git
cd AeraChat
```

---

### 2️⃣ Setup Backend

```bash
cd server
npm install
```

Create a `.env` file inside the `server` folder and add:

```env
MONGODB_URI = "your_mongodb_uri"
JWT_SECRET = "your_jwt_secret"
CLOUDINARY_CLOUD_NAME = "your_cloudinary_cloud_name"
CLOUDINARY_API_KEY = "your_cloudinary_api_key"
CLOUDINARY_API_SECRET = "your_cloudinary_api_secret"
```

Start the backend server:

```bash
npm start
```

---

### 3️⃣ Setup Frontend

```bash
cd client
npm install
npm run dev
```

---

### 4️⃣ Open in Browser

```
http://localhost:5173
```
